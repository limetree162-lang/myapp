# 📊 포트폴리오 정리 (Portfolio Organizer)

내 자산을 자동으로 정리하고 목표 배분에 맞춰 **리밸런싱**을 제안해 주는 웹 앱입니다.
서버가 필요 없으며, 데이터는 사용하는 브라우저에만 저장됩니다.

## 주요 기능

- **자산 등록/수정/삭제** — 종목명, 자산군, 평가금액 입력
- **현재 배분 시각화** — 자산군별 비중을 도넛 차트로 표시
- **목표 배분 설정** — 자산군별 목표 비중(%) 입력
- **자동 리밸런싱** — 목표에 맞추기 위해 무엇을 얼마나 **매수/매도**해야 하는지 계산
- **목표 이탈도** — 현재 배분이 목표에서 얼마나 벗어났는지 한눈에 확인
- **균등 배분 자동설정** — 버튼 한 번으로 자산군을 균등 비중으로 채우기
- **내보내기/불러오기** — 데이터를 JSON 파일로 백업/복원
- **다크/라이트 모드**
- **로컬 저장** — 새로고침해도 데이터 유지 (localStorage), 서버 전송 없음

## 사용 방법

별도 설치가 필요 없습니다. `index.html`을 브라우저로 열기만 하면 됩니다.

```bash
# 브라우저로 바로 열기
open index.html          # macOS
xdg-open index.html      # Linux

# 또는 로컬 서버로 실행
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

### 사용 순서

1. **자산 추가** 버튼으로 보유 종목을 하나씩 등록합니다.
   (예: `삼성전자 / 국내주식 / 5,000,000`)
2. 오른쪽 도넛 차트에서 현재 자산 배분을 확인합니다.
3. 하단 표에서 각 자산군의 **목표 비중(%)**을 입력합니다. (합계 100%가 되도록)
   - 잘 모르겠다면 **균등 배분 자동설정** 버튼을 눌러 시작하세요.
4. **조정** 열에 목표 달성을 위한 매수/매도 금액이 자동으로 계산됩니다.

## 자산군 예시

국내주식 · 해외주식 · 채권 · 현금 · 부동산 · 암호화폐 · 기타
(자산군은 자유롭게 직접 입력할 수 있습니다.)

## 파일 구조

```
index.html   # 화면 구조
styles.css   # 스타일 (다크/라이트 테마 포함)
app.js       # 앱 로직 (자산 관리, 차트, 리밸런싱 계산, 저장)
```

## 기술

의존성 없는 순수 HTML/CSS/JavaScript. 빌드 도구·프레임워크·인터넷 연결이 필요 없습니다.
